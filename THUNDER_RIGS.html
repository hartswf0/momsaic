AI API

Keys are stored in your browser only. With OpenAI, calls go directly to the official Responses API; a browser-stored key can be inspected, so use a temporary or restricted key.

AI BOTS

Reflex bots drive, chase, hunt and ram using local logic (no API). Architect bots also build forts, walls, towers and ramps — using your AI key when one is set (AI → API), falling back to prefab structures otherwise. In a room, only the host simulates bots; everyone sees them.

DRIVE

  • WASD / left joystick to drive
  • SHIFT or BOOST to nitro
  • SPACE or JUMP to hop
  • F or FIRE to shoot lasers

BUILD

  • BUILD toolbar places parts at the camera focus
  • Tap a part to open the inspector: resize, move, rotate, animate, clone, delete
  • GRAPH opens the scene tree — expand assemblies, select/lock/delete children. LK marks a piece indestructible (lasers and rams pass damage through it)

AI

  • AI → BUILD: describe a structure, an LLM drafts it, you place & commit
  • AI → API: set a key (OpenAI/Gemini/Anthropic/custom)

BOTS & COMBAT

  • MULTI → BOTS: add REFLEX (local AI) or ARCHITECT (API builder) bots
  • Architect bots build forts/walls/towers/ramps Fortnite/Rust-style
  • Lasers and high-speed rams demolish rival & bot rigs; destroyed rigs respawn
