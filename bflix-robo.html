B·FLIX Operations Manual
Computational Celluloid
Doctrine

Think in discrete blocks, trajectories, holds, and shifts. The blur belongs to the viewer, not the script. Mount video — annotate it — export it.

<> Script Mode

Open via Script button or bottom nav. Write BEFLIX-64 macros, hit Execute Script. Script ink is layered on top of any mounted video.

Grid

64 cols × 36 rows. Origin 0,0 = top-left. Values 0–7. 0 = transparent.

Palette: 0 CLR · 1 BLK · 2 RED · 3 GRN · 4 BLU · 5 YLW · 6 WHT · 7 INK

Commands
CLR v — fill entire matrix to v
PNT x y w h v — paint rectangle
LIN x1 y1 x2 y2 v — draw line
REC n — record matrix as n frames
SHF dx dy n — shift matrix over n frames
C text — comment (ignored)
Example — red box sliding right for 20 frames
C Red box, slides right
CLR 0
PNT 0 14 8 8 2
REC 1
SHF 3 0 20
✏ Tools Mode

Draw directly on the active frame. Pen and Box strokes are echoed to the script panel automatically.

Drawing Tools
Pen — freehand stroke (echoes to script)
Box — drag rectangle (echoes to script as PNT)
Circ — draw ellipse (sketch only, no echo)
Fill — flood fill a region (sketch only)
Erase — clear pixels to transparent
Controls
Color swatches — select palette color
Size slider — brush/line pixel size
Grid button — toggle 64×36 grid overlay
+ — add blank frame after current
⧉ — clone current frame
▣ Media Mode
Mounting
Mount — load image or video file
Camera — capture single frame from camera
Sample — load 24 test gradient frames
Video — extracts up to 300 frames at 12fps. Script loops to match video length (or vice versa).
Blend Modes
Overlay — script ink over video
Underlay — video over script ink
Multiply — darken blend
Screen — lighten blend
Burn — high-contrast darken
▶ Playback
Controls
Play / Stop — toggle animation loop at 12fps base
◀◀ / ▶▶ — cycle speed: 0.25× 0.5× 1× 1.5× 2× 3× 4×
← → arrows — step frames when stopped
Space — play/stop toggle (keyboard)
Swipe left/right — step frames (mobile)
Phosphor Trail
During playback a 16% ghost of the previous frame bleeds through — the optical persistence effect. Clears on stop.
